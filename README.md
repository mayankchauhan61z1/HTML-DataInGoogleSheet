# Contact Form with Google Sheets Integration

A simple and responsive Contact Form built using **HTML** and **CSS** that stores submitted data directly into **Google Sheets** using the **SheetDB API**.

## 🚀 Features

- Responsive contact form
- Clean and modern UI
- HTML5 form validation
- Stores data directly in Google Sheets using SheetDB
- Reset button to clear all fields

## 📂 Project Structure

```
project-folder/

```

## 🛠 Technologies Used

- HTML5
- CSS3
- SheetDB API
- Google Sheets

## 📋 Form Fields

- Name
- Email
- Contact Number
- Message

## ⚙️ How It Works

1. User fills out the contact form.
2. The form sends data using the POST method.
3. SheetDB receives the data.
4. SheetDB inserts the data into the connected Google Sheet automatically.
